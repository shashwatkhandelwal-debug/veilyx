01 — Quick Start

Up and running in 3 steps

1
Register your company
Get your API key in one call. No dashboard, no account creation.
bash 
curl -X POST https://web-production-fe8772.up.railway.app/company/register \
  -H "Content-Type: application/json" \
  -d '{"company_name": "YourApp"}'
response
{
  "company_id": "uuid",
  "company_name": "YourApp",
  "api_key": "your_api_key_here"  ← save this
}
2
Install the SDK
Drop the SDK into your Android or iOS project.
// build.gradle
implementation 'com.veilyx:sdk:1.0.0'
// Podfile
pod 'Veilyx', '~> 1.0'
npm install veilyx-react-native
3
Verify a user
Fetch a nonce, request a proof on-device, verify it server-side.
// 1. fetch nonce from your backend
val nonce = fetchNonce()

// 2. request proof on-device
val proof = veilyx.requestProof(
    companyName = "YourApp",
    checks = listOf("age_above_18"),
    nonce = nonce,
    aadhaarXml = xmlString
)

// 3. verify → result.valid == true
val result = api.verify(proof)
// 1. fetch nonce from your backend
let nonce = await fetchNonce()

// 2. request proof on-device
let proof = try await veilyx.requestProof(
    companyName: "YourApp",
    checks: ["age_above_18"],
    nonce: nonce,
    aadhaarXml: xmlString
)

// 3. verify → result.valid == true
let result = try await api.verify(proof)
// 1. fetch nonce
const { nonce } = await fetch(`${BACKEND_URL}/nonce`, {
  headers: { 'X-API-Key': API_KEY }
}).then(r => r.json());

// 2. request proof on-device
const proof = await Veilyx.requestProof({
  companyName: 'YourApp', checks: ['age_above_18'],
  nonce, aadhaarXml
});

// 3. verify → result.valid == true
const result = await fetch(`${BACKEND_URL}/verify`, {
  method: 'POST',
  headers: { 'X-API-Key': API_KEY },
  body: JSON.stringify(proof)
}).then(r => r.json());
02 — Authentication

API Key

All authenticated endpoints require your API key in the request header.

header
X-API-Key: your_api_key_here
⚡ Your API key is tied to your company name. The requested_by field in every proof must exactly match your registered company name.
